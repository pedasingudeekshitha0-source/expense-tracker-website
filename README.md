# 💰 Expense Tracker Website

A responsive Expense Tracker web application built using HTML, CSS, and JavaScript.

This project helps users track daily expenses, calculate totals, and manage spending efficiently through a simple and user-friendly interface.

---

## 🚀 Features

- Add income and expenses
- Automatic balance calculation
- Expense history tracking
- Clean and responsive UI
- Beginner-friendly project structure
- Fast and lightweight application

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript
